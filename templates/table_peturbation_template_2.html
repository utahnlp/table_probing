Table Comprehension Instructions (Click to Hide)

In this task, we will ask you to make inferences about the information present in a sentence.

You will be given a statement and a table. You have to select the rows of the table (you can select one or more than one, or none at all) that relate to the information presented in the sentence. i.e: ask yourself, could the information in a row prove or contradict the sentence? If so, select that row.

The following examples clarify the instructions. Please go over these examples carefully.
